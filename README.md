# Desafio de Projeto: O Dia a Dia de um QA - Testes Manuais na Amazon

Este repositório contém a entrega final do bootcamp da DIO, focada na prática de testes manuais funcionais. O projeto consistiu em planejar, documentar e executar testes em um cenário real de e-commerce (Amazon.com.br).

## 🚀 O que foi desenvolvido:
- **Plano de Fluxo de Trabalho:** Definição do ciclo de vida do bug (do Novo ao Fechado).
- **User Stories:** Criação de histórias de usuário com critérios de aceite para Busca e Carrinho.
- **Mapa Mental:** Mapeamento visual de cenários positivos, negativos e de interface.
- **Casos de Teste:** Execução de testes utilizando as técnicas **Step-by-Step** e **BDD (Behavior Driven Development)**.
- **Relatório de Evidências:** Documentação detalhada com capturas de tela dos resultados esperados vs. obtidos.
